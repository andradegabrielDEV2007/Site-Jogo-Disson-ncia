Site oficial de apresentação do projeto Dissonancia.
Desenvolvido para construir a identidade do jogo, apresentar suas principais ideias e criar um primeiro contato com o público.

Sobre:
Dissonancia é um site institucional criado para divulgar o universo do jogo, destacar suas mecânicas e fortalecer a proposta narrativa.

O foco do projeto foi trabalhar:

 - Estruturação semântica em HTML
 - Organização modular de CSS
 - Implementação de backend com Flask
 - Estruturação de rotas
 - Integração com banco de dados utilizando SQLAlchemy

Tecnologias utilizadas:
 - HTML5
 - CSS3
 - PYTHON
 - FLASK
 - SQLALCHEMY
